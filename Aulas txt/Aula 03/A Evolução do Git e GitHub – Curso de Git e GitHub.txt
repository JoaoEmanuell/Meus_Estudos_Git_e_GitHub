O git foi criado em 2005 apos o criador do BitKeeper retirar varios recursos de versionamento da versão comunidade, dessa forma o criador do linux ficou iritado e criou o git

O git era um softwares distribuido, open source, foi feito em 10 dia e além disso foi focado em performace, uma vez que os outros versionadores demoravam muito para fazer um commit, o git demorava muito pouco

Git pode siginificar 4 coisas uma vez que o criador original nunca disse o real siginificado do termo:

Três letras do alfabeto

Temioso, cabeça-dura, pensa que sempre está certo

Global information tracker
